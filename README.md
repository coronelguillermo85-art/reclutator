## 🎮 El Concepto
En esta expansión, el desafío escala. Represento de forma satírica y técnica las barreras más comunes del mercado laboral actual:
* **Excusas en modo ráfaga:** "Buscamos un junior con 10 años de experiencia" o "Te vamos a tener en cuenta".
* **Resistencia extrema:** Un jefe final que requiere estrategia, reflejos y, por supuesto, una lluvia de CVs precisos.

## 🛠️ Especificaciones Técnicas
* **Motor:** JavaScript puro (Vanilla JS) y HTML5 Canvas API.
* **Mecánicas:** Sistema de colisiones avanzado, gestión de fases de jefe y lógica de proyectiles.
* **Optimización:** Código liviano sin dependencias externas, diseñado para ejecución inmediata en cualquier navegador.
